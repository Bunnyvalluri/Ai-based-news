NLP Pipeline & REST API Reference

Our AI models utilize a state-of-the-art Natural Language Processing pipeline to analyze text patterns, linguistic markers, and semantic consistency to detect disinformation.

1 The NLP Pipeline

Before inference, every text string passes through a rigorous 5-step preprocessing sequence to ensure high accuracy and remove linguistic noise.

data_object

Tokenization

Splitting text into individual words.

format_list_bulleted

Lowercasing

Standardizing text case sensitivity.

content_cut

Lemmatization

Reducing words to base forms.

filter_list

Stop-words

Removing common noise words.

grid_3x3

Vectorization

Conversion to numerical data.

2 REST API Reference

POST /api/v1/predict
v2.1.0-stable

Analyze a single text sample for potential fake news content. Returns a probability score and top contributing features.

Request Body
{
  "text": "The latest report shows that...",
  "model": "fakenews-detector-pro-v1",
  "explain": true
}
Implementation Example (Python)
import requests

url = "https://api.fakenewsai.com/v1/predict"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "text": "Viral post claiming miracle cures...",
    "model": "latest"
}

response = requests.post(url, json=data, headers=headers)
print(response.json())

3 Understanding Metrics

Precision

The ability of the classifier not to label as disinformation a post that is actually authentic.

98.4%

Recall

The ability of the classifier to find all the disinformation instances in the dataset.

96.1%

F1 Score

The harmonic mean of precision and recall, providing a balanced measure of performance.

97.2%
