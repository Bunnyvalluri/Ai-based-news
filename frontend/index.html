      Pipeline

      How TruthLens Works

      A 5-stage NLP + ML pipeline processes every article in under 3 seconds.

      01

      Input & Validation

      User submits news text via the web UI or API. Input undergoes validation: minimum length, max word count, and format checks.

      ‚Üí
      02

      NLP Preprocessing

      Text is normalized: lowercased, punctuation stripped, tokenized, stop-words removed, then lemmatized via NLTK WordNet.

      ‚Üí
      03

      TF-IDF Vectorization

      Tokens are converted to numerical vectors using TF-IDF with 50,000 unigram+bigram features ‚Äî fitted only on training data to prevent leakage.

      ‚Üí
      04

      ML Classification

      The feature vector is classified by the best-trained model (LR, NB, RF, or SVM). Probability scores produce the confidence rating.

      ‚Üí
      05

      Result & Explanation

      A REAL/FAKE verdict with confidence score and top contributing keywords are returned to the UI in under 3 seconds.
